# API Examples

This folder contains example requests and responses for the Sohar Festival API endpoints.

## Structure

Each API endpoint has its request and response stored in separate JSON files:

- `*-request.json` - Example request body
- `*-response.json` - Example response body

## Folders

- **auth/** - Authentication endpoints (OTP, login, profile)
- **events/** - Event listing and management
- **tickets/** - Ticket pricing and purchases
- **restaurants/** - Restaurant listings
- **heritage-villages/** - Heritage village information
- **cultural-workshops/** - Workshop listings and registration
- **traditional-activities/** - Traditional activity listings
- **craft-demonstrations/** - Craft demonstration schedules
- **photo-spots/** - Photo location recommendations
- **map-locations/** - Map points of interest
- **notifications/** - User notifications
- **payments/** - Payment processing
- **emergency-contacts/** - Emergency contact information
- **announcements/** - Festival announcements
- **app-settings/** - App configuration settings
