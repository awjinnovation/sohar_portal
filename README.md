# Sohar Festival Mobile Application

A comprehensive Flutter mobile application for the Sohar Festival, featuring bilingual support (Arabic/English), event management, ticketing, and cultural experiences.

### Core Features
- **Bilingual Support**: Full Arabic and English localization with RTL/LTR text direction
- **Authentication**: Sign in, create account, and user management
- **Event Management**: Browse, search, and filter festival events by category
- **Ticketing System**: Purchase tickets and display QR codes for entry
- **Interactive Map**: Navigate the festival venue with location markers
- **Restaurant Directory**: Discover dining options with ratings and menus
- **Healthcare Services**: Emergency contacts and first aid locations
- **Heritage Villages**: Explore cultural experiences and traditional crafts

### Key Screens
1. **Home Screen**: Festival overview with quick access to main features
2. **Events Schedule**: Calendar and list views of all festival activities
3. **Ticketing**: Buy tickets and manage purchased tickets with QR codes
4. **Interactive Map**: Festival venue navigation
5. **Restaurants**: Browse and discover dining options
6. **Healthcare**: Emergency services and medical facilities
7. **Heritage Villages**: Cultural experiences and activities
